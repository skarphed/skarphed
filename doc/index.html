Documentation of skarphed
Hello there!
My name is Saiko Takashi and i work at skarphed headquarters as a systems administrator. I have a huge experience in using skarphed for webmanagement, since i know the guy who wrote it. As we all know, programmers hate writing documentations, so I wrote this usermanual for all newcomers and heavy users who want to learn more. I hope, it will be useful. Have fun with skarphed :)

If you are new to skarphed, you probably want to read our FAQ-Section to inform yourself about the project and its purpose.

If you want to learn how to use skarphed and it's components to build your web-environment, you want to read the Users Manual

If you want to help developing skarphed or skarphed-Modules or you simply want to understand how skarphed works internally, please step ahead to read the Developers Manual
