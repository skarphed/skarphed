Overview (getting started)

This section will teach you what components a working scoville environment consists of and how to set it up.

Components

A basic scoville installation consists of a Firebird-Database that will be used to store the persistent data of your scoville-managed websites or webapplications, a scoville-repo which is a repository that provides you with modular extensions (modules) and website-designs (templates) and finally the scoville-core itself, which is the application that runs on the server and answers your audience's HTTP-request.
